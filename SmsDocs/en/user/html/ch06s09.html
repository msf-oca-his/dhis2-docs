6.9. Translate data sets

DHIS 2 provides functionality for translations of database content like data elements, data element groups, indicators, indicator groups, validation rules and more. You can translate these elements to any number of locales. A locale represents a specific geographical, political, or cultural region.

Tip

To activate a translation, go to Apps > Settings > Appearance and select a language.

  1. Click Apps > Maintenance > Data set > Data set.

  2. Select the object type you want to translate, for example indicator or indicator group.

  3. In the list, find the object you want to translate.

  4. Click the options menu and select Translate.

  5. Select a locale.

  6. Enter the required information.

  7. Click Close.
