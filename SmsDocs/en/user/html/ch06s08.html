6.8. Display details of data sets

  1. Click Apps > Maintenance > Data set > Data set.

  2. Select the object type you want to display, for example indicator or indicator group.

  3. In the list, find the object you want to view.

  4. To display all metadata about the selected object, click the options menu and select Show details.
