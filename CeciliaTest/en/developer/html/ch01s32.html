1.32. Data set reports

Data set reports can be generated trough the web api using the /datasetreport resource. This resource generates reports on data set and returns the result in the form of a HTML table. The request to /datasetreport can or must contain the following parameters:

Table 1.62. Accepted parameters of /datasetreport

ParameterDescriptionTypeRequired
dsData set to create the report from.Data set UIDyes
pePeriod to create the report from.ISO Stringyes
ouOrganisation unit to create the report fromOrganisation unit UIDyes
dimensionDimensions to be used as filters for the reportOne or more UIDsno
selectedUnitOnlyWhether to use captured or aggregated dataBooleanno


/datasetreport only has one accepted method: GET. An example request to retrieve a report for a dataset A and orgunit O for 2015 could be:

GET /api/datasetreport?ds=A&ou=O&pe=2015&selectedUnitOnly=false
