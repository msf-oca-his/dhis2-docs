6.4. Assign a data set to organisation units

  1. Click Apps > Maintenance > Data set > Data set.

  2. In the list, find the data set you want to assign to organisation units.

  3. Click the options menu and select Assign to organisation units.

  4. In the organisation unit tree, select the organisation units you want to assign the data set to. The selected organisation units are marked in yellow.

  5. Click Close.

Your data set is now ready to be used within the Data entry app for the organisation units that you have assigned and for periods according to your selected frequency (period type).
