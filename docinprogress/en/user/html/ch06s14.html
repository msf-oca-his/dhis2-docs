6.14. Edit multiple data sets at once

With the Data set assignment editor you can add and remove multiple data sets to organisation units at once.

  1. Click Apps > Maintenance > Data set assignment editor.

  2. Select an organisation unit.

    A table appears with data sets as columns and child organisation units as rows.

  3. Assign or unassign data sets by clicking the corresponding icons in the table.

    Your changes are saved automatically.

    Tip

    If you want to assign or unassign an organisation unit to all data sets, select or clear the check box next to the organisation unit.
