5.10. Create category option groups and group sets

Category options can be grouped and classified using category option groups. Category option groups can be included in category option group sets.

The main purpose of the these entities is to be able to add more dimensionality to your captured data for analysis in applications as for example the Pivot Table or Data Visualizer.

Example 5.4. Category options groups and group sets best practice

In a system, data is collected by "projects", where projects is modeled as category options. The system must be abpe to do data analysis according to which donor supports the project. In this case, a category option group set called "Donor" can be created. Each donor can be created as a category option group, where each category option / project is put in the appropriate group. In data analysis applicationss, the "Donor" group set will appear as a data dimension, while each donor appear as dimension items, ready for inclusion in reports.


To create a category option group:

  1. Click Apps > Maintenance > Data Elements > Category Option Group.

  2. Click the yellow plus button.

  3. Fill in the form:

    1. Name: Each data element must have a unique name

    2. Short name: Define a short name for the data element.

    3. Code

    4. Data dimension type

  4. Select Category options and assign them.

  5. Click Save.

To create a category option group set:

  1. Click Apps > Maintenance > Data Elements > Category Option Group Set.

  2. Click the yellow plus button.

  3. Fill in the form:

    1. Name

    2. Description

    3. Data dimension

    4. Data dimension type

  4. Select Category option groups and assign them.

  5. Click Save.
