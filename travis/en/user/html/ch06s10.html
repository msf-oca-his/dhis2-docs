6.10. Download data sets in PDF format

You can download a data set/a data form in PDF format for offline data entry.

Get PDF for Data Entry: Download a PDF file which can be used for offline data entry.

  1. Click Apps > Maintenance > Data set > Data set.

  2. In the list, find the object you want to download.

  3. Click the options menu and select Get PDF for data entry.
