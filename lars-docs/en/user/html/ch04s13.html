4.13. Move organisation units within a hierarchy

You can move organisation units within in the hierarchy by changing the parent of a selected organisation unit.

  1. Click Apps > Maintenance > Organisation units > Hierarchy operations.

  2. Select the organisation units you want to move:

    1. In the hierarchy tree to the left, select the organisation unit you want to move.

    2. In the hierarchy tree to the right, select where you want to move the organisation unit to.

    3. Repeat selection for all the organisation units you want to move.

  3. Click Move x organisation units, where x stands for the number of organisation units you have selected.

    Your changes are immediately reflected in the left-hand side hierarchy tree.
