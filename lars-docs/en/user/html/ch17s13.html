17.13. Using place search

The place search function allows you to search for almost any location or address. The place search is powered by the Mapzen mapping platform. This function is useful in order to locate sites, facilities, villages, towns and more on the map.

  1. Click the magnifier icon on the left side of the screen.

  2. Enter your search term.

  3. Select from the list of matching locations which appear as you type. A pin will indicate the location on the map.
