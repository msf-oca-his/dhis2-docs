5.9. Create a category option group

You can goup and classify category options by using category option groups. The main purpose of the category option group set is to add more dimensionality to your captured data for analysis in for example the Pivot table or Data Visualizer apps.

Example 5.3. Category options groups and group sets

In a system, data is collected by "projects", where projects is modeled as category options. The system must be able to analyse data based on which donor that supports the project. In this case, create a category option group set called "Donor". Each donor can be created as a category option group, where each category option / project is put in the appropriate group. In the data analysis applicationss, the "Donor" group set will appear as a data dimension, while each donor appear as dimension items, ready to be included in reports.


To create a category option group:

  1. Click Apps > Maintenance > Data elements > Category option group.

  2. Click the yellow plus button.

  3. Fill in the form:

    1. Name

    2. Short name: Define a short name for the data element.

    3. Code

    4. Data dimension type

  4. Select Category options and assign them.

  5. Click Save.
